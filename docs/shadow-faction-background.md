# Shadow Faction Background

影族并非单纯亡灵，而是被雾潮与旧战场残响扭曲后的边境军团。它们常在沼泽、古墓与沉没哨站附近聚集，依靠残存执念维系松散秩序。

首批四个兵种围绕“消耗战”和“处决窗口”设计：

- `shadow_skeleton` 作为低成本前排，用 `death_resilience` 扛住第一次溃散。
- `shadow_wraith` 依靠 `armor_pierce` 威胁高防守军。
- `shadow_hexer` 通过禁魔与削弱打乱敌方节奏。
- `shadow_death_knight` 负责在残局用 `life_drain` 收割并续航。

当前落地图点选在 `phase1-map-objects-murkveil-delta.json`，将影族定义为雾港外围的中立威胁，便于后续继续扩展地图剧情与阵营专属招募点。
